裡面有兩組不同的作品，第一個是 Mine_sweeper 踩地雷，六月初就寫完了，但我認為有點普通，所以就在想可以加甚麼，想到的延伸為以 database 來儲存遊戲資料，但因為踩地雷一局遊戲太短，沒啥好儲存的，所以換成數獨。

 # 程式碼中我都有寫中文註解，如果是亂碼可能要請老師以其他方式開啟檔案

 # 踩地雷

只有一個檔案，會以亂數去生成一個特定大小的地圖，並以亂數放入特定個數地雷，再來程式會計算每格四面八方有幾個格子，
然後就會印出地圖，並開始遊戲，玩家可以選擇要開啟的格子，程式會判斷要給的資訊，直到玩家勝利或失敗，沒有影片是因為空間不足。


 # Sudoku 數獨

以資料庫系統為主，可以寫入檔案、修改檔案、刪除檔案，查找檔案，
最後也可以進入遊戲，隨時可以檢查答案是否正確，遊戲中還可以存檔，下次啟動程式後讀檔接著玩。
有三個檔案， SudokuData 內為一個 class ，Sudoku_function放了幾個外部函式，
最後是 Sudoku_main ，為主程式，由其呼叫其他函式。
